# Infographic Prompt Pattern

## Use Case

For visual explanations, process diagrams, training graphics, data summaries, and educational layouts.

## Prompt Structure

```text
Create an infographic about [topic].

Audience:
[who will read it]

Goal:
[what they should understand or do]

Information structure:
1. [section]
2. [section]
3. [section]

Visual hierarchy:
[title, key numbers, steps, labels]

Style:
[clean, editorial, technical, classroom-friendly, etc.]

Output:
[ratio, size, file type]

Constraints:
[language, readability, no clutter, source notes]
```

## Checklist

- Is the reading order clear?
- Are labels short?
- Is the language specified?
- Is the format specified?
- Is there enough whitespace?
