# Poster Prompt Pattern

## Use Case

For posters, event graphics, narrative key visuals, and social media hero images.

## Prompt Structure

```text
Create a poster for [project / person / event].

Core concept:
[one-sentence concept]

Visual metaphor:
[symbol, scene, or narrative device]

Composition:
[layout, focal point, foreground/background, hierarchy]

Subject:
[person / object / brand / animal details]

Typography:
[title, subtitle, placement, language, hierarchy]

Style:
[visual style, texture, lighting, color, era]

Output:
[aspect ratio, size, format]

Constraints:
[must preserve / must avoid / logo rules / face rules]
```

## Checklist

- Is the subject clear?
- Is the title readable?
- Is the output ratio specified?
- Are face/logo preservation rules included?
- Are forbidden elements listed?
