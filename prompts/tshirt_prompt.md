# T-Shirt Prompt Pattern

## Use Case

For T-shirt graphics, team apparel, event merch, and print-ready illustration concepts.

## Prompt Structure

```text
Create a T-shirt graphic for [brand / team / event].

Core idea:
[one-sentence concept]

Graphic system:
[central icon, mascot, typography, badge, emblem, or scene]

Print style:
[screen print, vector-like, distressed, embroidered look, etc.]

Colors:
[limited palette and shirt color]

Typography:
[text content, placement, hierarchy]

Constraints:
[print limitations, no gradients, no tiny details, preserve logo rules]

Output:
[transparent background / mockup / front chest / back print]
```

## Checklist

- Can it print cleanly?
- Are colors limited?
- Is text legible?
- Does it work without a background?
- Are brand/logo rules respected?
