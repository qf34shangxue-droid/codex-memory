# design word2

## Purpose

Use this file to record the concept system, naming logic, visual metaphors, and prompt rules for `design word2`.

`design word2` is a visual-metaphor-driven system. It treats the word and the image as a single visual sentence.

## Core Identity

- Name: design word2
- Role: High-concept graphic design system driven by visual metaphor
- Tone: Strong, intelligent, refined, poster-like, concept-first
- Visual keywords: visual metaphor, word-image sentence, single strong concept, advanced graphic design, semantic compression
- Avoid: multiple weak ideas, literal collage, decorative typography, unrelated background images, low-level gimmicks

## Naming And Meaning

- Word / phrase: Defined per project, often from a nickname, event, person, or poetic phrase.
- Literal meaning: The surface meaning should be understood first.
- Deeper metaphor: The design must translate the phrase into a visual metaphor, not merely display the text.
- Emotional direction: One clear emotional direction per work: heroic, ironic, elegant, intense, playful, ceremonial, etc.

## Visual Rules

- Composition: Use one dominant concept and make every visual decision serve it. The viewer should understand the visual sentence quickly.
- Typography: Text and graphics should be fused. The word may become a scene, symbol, body, object, or structural frame.
- Color: Choose a palette that supports the concept and feels designed rather than decorative.
- Texture: Use texture sparingly; it should reinforce atmosphere, material, or narrative.
- Character / symbol treatment: Characters, animals, brands, or objects should be transformed into symbols with clear semantic connection to the phrase.

## Core Rules

- Visual metaphor drives the design.
- Image and word together form one visual sentence.
- Each work should have a single strong concept.
- The style should feel like advanced contemporary graphic design.
- Avoid explaining too many ideas at once; compress the idea into one memorable visual structure.

## Example Direction

For a phrase such as `龟山之王`, the system should first analyze:

- What does the phrase literally mean?
- What is the implied character or myth?
- What symbols carry the meaning?
- How can the text and symbols become one image?
- What ratio, hierarchy, and typography best serve the concept?

## Prompt Notes

- Reusable opening: Create a high-concept `design word2` graphic where the phrase and image form one unified visual sentence.
- Required constraints: Use one strong visual metaphor; integrate typography and image; define composition, proportion, hierarchy, and symbolic logic clearly.
- Negative prompts: generic poster, unrelated illustration, scattered ideas, decorative text, overcomplicated symbolism, low-quality typography
