# design word

## Purpose

Use this file to record the core concept, tone, visual language, and recurring rules for the `design word` persona or series.

`design word` is a typography-led visual design system. The word itself is the protagonist, not a caption attached to an image.

## Core Identity

- Name: design word
- Role: Typography-centered visual concept system
- Tone: Conceptual, graphic, precise, visually intelligent
- Visual keywords: typography as protagonist, deep metaphor, composition discipline, word-image fusion
- Avoid: decorative text, generic poster layout, image-first composition where the word becomes secondary, shallow literal illustration

## Style Notes

- Composition: The word should dominate the composition. Supporting forms, images, or symbols must serve the word's semantic structure.
- Typography: Type is the main visual material. Letterforms may become containers, structures, paths, silhouettes, or symbolic objects.
- Color: Use color to strengthen hierarchy and meaning rather than decoration.
- Texture: Texture may be used when it reinforces the concept, material feeling, or emotional tone.
- Symbol system: Symbols should emerge from the word's meaning, pronunciation, cultural association, or metaphor.

## Core Rules

- The font / word is the absolute main character.
- Every design should contain a deeper metaphor, not just a literal illustration.
- Composition rules matter: hierarchy, balance, rhythm, and negative space should be deliberate.
- Text and image should form one integrated visual statement.
- The final result should still read as a designed word, even when viewed quickly.

## Prompt Notes

- Reusable opening: Create a typography-led `design word` visual where the word itself is the central subject.
- Required constraints: Make the typography the dominant element; build the visual metaphor from the meaning of the word; keep the composition disciplined and graphic.
- Negative prompts: generic poster, decorative caption, random illustration, cluttered composition, weak typography, low-level literalism
